Hugo搭建自己的博客教程

Posted by     "YinChao" on Wednesday, October 19, 2022

Hugo 搭建自己的博客

模板选择

https://themes.gohugo.io/themes/hugo-theme-cleanwhite/

搭建博客

以下命令均在Ununtu20.04上进行操作

  1. 创建项目： hugo new site website
  2. 从github中拷贝主题 
    cd themes
git clone https://github.com/zhaohuabing/hugo-theme-cleanwhite.git
  3. 拷贝完成后，拷贝主题到website目录 
    cp -r hugo-theme-cleanwhite/exampleSite/** ../
  4. 注意，当前处于themes目录，可以用cd ..返回上级目录
  5. 本地运行：hugo serve
  6. 打开浏览器输入:localhost:1313查看效果

将博客内容挂到github上

  • 进入你的github账号，如果你的用户名叫做zhangsan,则新建一个名字叫做zhangsan.github.io的仓库
  • 在章节[搭建博客]中，进入website目录，输入hugo命令，会自动在该目录下新建public目录。
  • 将该目录中的内容全部上传到新创建的zhangsan.github.io仓库
  • 在浏览器中输入https://zhangsan.github.io即可以访问你自己的站点了
  • 至此，远程访问你的整个博客站点就完成了。

如何将public目录的内容上传到zhangsan.github.io仓库

首次上传，本地仓库没有关联到远程仓库，所以需要git remote add origin ...

进入到website/public目录下，依次输入以下命令
```
git init
git add .
git commit -m "Hugo build my blogs"
git remote add origin git@github.com:zhangsan/zhangsan.github.io.git
git push -u origin master
```
命令全部执行成功，即可通过网址`https://zhangsan.github.io`访问了。

后续上传，只需要执行正常的git操作进行提交即可，将public目录中的内容上传到目的仓库即可

自定义博客

由于执行了上面的命令cp -r hugo-theme-cleanwhite/exampleSite/** ../，所以显示的是默认的博客模板。自己需要对博客内容进行修改。 主要修改可能有下面几点:

  • 修改config.toml文件
  • 删除 website/content目录下的子目录中的一些 “*.md"文档
  • 修改或者删除 website/static/img目录下的一些图片。
  • 具体要怎么修改，可以先将demo网站运行起来，根据自己的需要来进行修改。

注意点

  • 本博客站点的下载地址：https://github.com/luweiqianyi/website.git
  • 通过git clone同步到本地时，需要新增或者修改文章均在目录website\content\post中修改
  • 修改结束，在目录website下运行hugo serve，并访问路径localhost:1313看看刚才的修改是否生效
  • 如果生效，将public整个目录上传到https://github.com/luweiqianyi/luweiqianyi.github.io.git地址，作为在github上搭建博客站点的内容来源。
