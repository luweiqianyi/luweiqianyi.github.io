Windows下部署etcd集群

前提

Windows下已经安装etcd。

新建项目

项目暂且命名为etcd_cluster,以该名字创建目录。 项目根目录路径：F:\\etcd_cluster 在该目录下创建下面的目录结构和对应的文件。

etcd_cluster
├── etcd1/                      // etcd1节点相关目录
│   └── data/                   // 存放etcd1节点的运行数据
│   └── etcd1_conf.yml          // 启动etcd1节点的配置信息
├── etcd2/                      // etcd2节点相关目录
│   └── data/                   // 存放etcd2节点的运行数据
│   └── etcd2_conf.yml          // 启动etcd2节点的配置信息
├── etcd3/                      // etcd3节点相关目录
│   └── data/                   // 存放etcd3节点的运行数据
│   └── etcd3_conf.yml          // 启动etcd3节点的配置信息 
├── starEtcd1.bat               // 启动etcd1的脚本文件
├── starEtcd2.bat               // 启动etcd2的脚本文件
├── starEtcd3.bat               // 启动etcd3的脚本文件

编写三个etcd节点的配置信息

etcd1_conf.yml

name: etcd1
data-dir: ./data    # 设置数据目录为当前目录的data目录
listen-client-urls: http://127.0.0.1:12379
advertise-client-urls: http://127.0.0.1:12379
listen-peer-urls: http://127.0.0.1:12380
initial-advertise-peer-urls: http://127.0.0.1:12380
initial-cluster: etcd1=http://127.0.0.1:12380,etcd2=http://127.0.0.1:22380,etcd3=http://127.0.0.1:32380
initial-cluster-token: etcd-cluster-1
initial-cluster-state: new

etcd3_conf.yml

编写启动etcd节点的bat文件

startEtcd1.bat

echo off
F:
cd F:\etcd_cluster
cd etcd1
etcd --config-file=etcd1_conf.yml
cmd

startEtcd2.bat

echo off
F:
cd F:\etcd_cluster
cd etcd2
etcd --config-file=etcd2_conf.yml
cmd

startEtcd3.bat

echo off
F:
cd F:\etcd_cluster
cd etcd3
